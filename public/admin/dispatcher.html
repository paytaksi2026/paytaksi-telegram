Dispetçer (Auto-Assign)

Parametrlər

OFF
İş prinsipi: Müştəri sifariş verən kimi server ən yaxın uyğun sürücünü təyin edir. Sürücü 20 saniyə (və ya burda seçdiyin) cavab verməsə, sifariş avtomatik növbəti sürücüyə ötürülür. Uyğun sürücü seçimi: online + approved + GPS son TTL içində + radius + paket icazəsi + balans limiti.

Qısa qeyd

• Auto-Assign ON olanda sürücü feed siyahısı görmür — yalnız ona təyin olunan sifariş popup kimi gəlir.
• Bu səhifə settings API üzərindən işləyir. Hər dəyişiklikdən sonra server dispatcher loop-u yenidən başladır.
• İstəsən gələcəkdə əlavə edərik: “Sifariş retry limiti”, “Ən yaxın N sürücü”, “Surge ilə prioritet”, “Reject səbəbləri”.
